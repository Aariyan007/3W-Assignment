## âœ¨ Features

### Authentication
- User signup and login using email and password
- User data stored securely in MongoDB
- Passwords hashed before storage

### Posts
- Users can create posts with **text**, **image**, or both  
- Public feed showing posts from all users
- Each post displays:
  - Username
  - Content
  - Like count
  - Comments

### Likes & Comments
- Users can like any post
- Users can comment on any post
- Usernames of users who liked or commented are stored and displayed
